## Instructions 
1. Check .toml file
```bash
$ poetry check
```
2. Verify poetry.lock
```bash
$ poetry lock --check
```
3. Install dependencies
```bash
$ poetry install
```
Alternatively, you can use pip
```bash
$ pip install -r requirements.txt
```
4. Run script
```bash
$ python main.py
```
